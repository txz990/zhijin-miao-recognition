基于DTW模板匹配的织金苗语小词表语音识别系统

本系统用于织金苗语小词表语音识别实验与演示，当前版本提供音频识别、历史记录、基础统计与模板重建能力。采用先进的语音处理技术，为少数民族语言研究提供有力支撑。

🎙️ 进入识别 📋 查看记录
💻 技术路线 VAD + MFCC + DTW
🔧 运行环境 Java 11 / Spring Boot
🎯 系统定位 论文配套实验系统
📊 初始数据 演示级初始化数据

系统用途

  • 用于小词表苗语语音识别流程演示与基础验证
  • 用于论文答辩阶段展示识别、记录与模板管理能力
  • 用于后续补充实验脚本与完整语料后的扩展研究

当前模块

  • 语音识别：上传 WAV 音频并返回识别结果或拒识结果
  • 历史记录：查看最近识别结果、距离、置信度与处理耗时
  • 统计分析：查看识别次数、平均置信度与基础统计
  • 模板管理：支持按样本或按词条重建模板

使用约束

  • 输入音频应为 WAV 格式，建议 16kHz、16bit、单声道
  • 当前仓库默认不内置论文完整实验语料与评测结果
  • 论文中的完整实验规模与指标需在单独导入语料后复现

推荐入口

🎙️ 语音识别 上传测试音频并查看识别结果 📊 统计分析 查看当前系统识别统计与运行概况 📋 历史记录 回看最近识别结果与拒识情况
